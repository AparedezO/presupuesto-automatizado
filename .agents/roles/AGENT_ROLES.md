# Roles De Agentes

## Product Spec Agent

Convierte necesidades del negocio en requisitos, especificaciones y criterios de aceptacion.

## Architecture Agent

Propone estructura tecnica, modelo de datos, integraciones y decisiones de arquitectura.

## Implementation Agent

Implementa funcionalidades siguiendo la especificacion aprobada y los patrones del proyecto.

## QA Agent

Valida que la implementacion cumpla criterios de aceptacion, pruebas y evidencia.

## Documentation Agent

Mantiene documentacion de uso, decisiones, cambios y entregables.
