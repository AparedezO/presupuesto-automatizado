# Trabajo Con Agentes

Esta carpeta contiene solo la coordinacion necesaria para que los agentes trabajen con trazabilidad y sin perder contexto.

## Estructura

- `roles/`: responsabilidades de cada agente usado en el desarrollo.
- `workflows/`: flujo de trabajo SDD aplicado al proyecto.
- `runs/`: registro de ejecuciones relevantes, decisiones tomadas y resultados.

## Principio

Los agentes no reemplazan la especificacion. La usan como contrato de trabajo.
